mycluster Springboard

Entrypoints

  • Kubernetes Dashboard
  • myapp info API
  • myapp randow API
  • myapp metric API
  • InfluxDB UI (configure influx server at http://localhost:31081)
  • Goldilocks
  • Prometheus
  • Grafana (use admin/operator to login)
  • Alertmanager

Feature Settings

The settings are controlled in the file config.sh
  • SLACK_ENABLE: no - controls if Slack is used for alerting. Slack account is required.
  • GRAFANA_CLOUD_ENABLE: no - controls if grafana-cloud is deployed. Grafana Cloud account is required.
  • GOLDILOCKS_ENABLE: yes - controls if goldilocks is deployed.
  • KUBERNETES_DASHBOARD_ENABLE: yes - controls if Kubernetes Dashboard is deployed.
  • KEDA_ENABLE: no - controls if keda is deployed.
  • NGINX: yes - controls if nginx is deployed
  • RESOURCEPATCH: yes - controls if resource settings are patched everywhere.
  • configuration is defined in ./config.sh
  • export KUBECONFIG=/home/klaus/.k3d/kubeconfig-mycluster.yaml
